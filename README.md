# ZapretRecode

Форк [zapret-discord-youtube](https://github.com/Flowseal/zapret-discord-youtube) с коллекцией из 43 готовых профилей для обхода DPI-блокировок. Каждый профиль это отдельная комбинация параметров winws, протестированная на разных провайдерах.

## Что это

Набор bat-файлов с разными настройками DPI desync методов. Вместо того чтобы самому подбирать параметры, просто запускаешь профили по очереди и смотришь какой работает.

## Отличия от оригинала

Это форк Flowseal/zapret-discord-youtube. Основное отличие - 43 готовых профиля вместо одного дефолтного.

Профили отличаются:
- Количеством повторов (repeats от 5 до 25)
- Методами обхода (fake, multisplit, fakedsplit, multidisorder)
- Паттернами fooling (ts, badseq, md5sig)
- Значениями autottl и cutoff
- Позициями split и seqovl
- Комбинациями методов


## Структура профилей

### Основные (Anti-RKN.01-30)

Профили с разными комбинациями параметров, от простых до сложных:

- **01-10**: fake с разным количеством повторов, базовые настройки
- **11-20**: fake + fakedsplit, разные паттерны (0x00, 0xFF, 0xAA)
- **21-30**: multisplit, multidisorder, продвинутые методы

### Альтернативные (Anti-RKN.ALT1-13)

Экспериментальные варианты с нестандартными комбинациями параметров

## Использование

### Быстрый старт

1. Распакуй архив
2. Запусти `Anti-RKN.07.bat` двойным кликом
3. Проверь работу Discord/YouTube

Если не работает - пробуй другие профили. Начни с меньших номеров (01-06), потом попробуй большие (15-30), потом ALT.

### Установка как сервис

1. Запусти `service.bat` от админа
2. Выбери пункт 1 (Install Service)
3. Выбери профиль из списка
4. Сервис запустится автоматически при загрузке Windows

### Остановка

Через диспетчер задач убей процесс winws.exe или:
```batch
taskkill /F /IM winws.exe
```

Для удаления сервиса используй пункт 2 в service.bat

## Требования

- Windows 10+
- Права администратора
- winws.exe и WinDivert (идут в комплекте)

## Что внутри

```
Anti-RKN/
├── Anti-RKN.01-30.bat    # Основные профили
├── Anti-RKN.ALT1-13.bat  # Альтернативные профили
├── service.bat           # Меню установки/диагностики
├── bin/
│   ├── winws.exe
│   └── *.bin             # TLS/QUIC паттерны
└── lists/
    ├── list-general.txt  # Домены для обхода
    ├── list-google.txt   # Google домены
    ├── list-exclude.txt  # Исключения
    ├── ipset-all.txt     # IP адреса
    └── ipset-exclude.txt # IP исключения
```

## Настройка списков

Добавь домены в `lists/list-general.txt` если нужно обходить блокировки на конкретных сайтах.

Добавь домены в `lists/list-exclude.txt` если какие-то сайты работают плохо (банки, госуслуги).

## Методы обхода

Профили используют разные DPI desync методы:

- **fake** - отправка поддельных пакетов
- **multisplit** - разделение с перекрытием последовательностей
- **fakedsplit** - комбинация fake и split
- **multidisorder** - разделение с изменением порядка
- **fooling** - дополнительные техники (ts, badseq, md5sig)

## Диагностика

Запусти `service.bat` и выбери пункт 4 (Run Diagnostics). Скрипт проверит:

- Конфликты с другими bypass (GoodbyeDPI и т.д.)
- Проблемные сервисы (Adguard, Killer, SmartByte)
- Настройки TCP timestamps
- Прокси
- VPN конфликты

## Проблемы

### Не работает после запуска

1. Проверь что winws.exe запущен в диспетчере задач
2. Попробуй другой профиль
3. Запусти диагностику через service.bat
4. Проверь что антивирус не блокирует winws.exe

### Низкая скорость

Используй профили с меньшим количеством повторов (01-10) или добавь проблемные домены в list-exclude.txt

### Конфликт с VPN

Запускай Anti-RKN после подключения VPN
